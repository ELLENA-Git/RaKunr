通过投稿您可以和所有人分享您所喜欢的一句话。 获得一句话的API，将一句话搬到自己的网站上。

### PHP
在模板的 `functions.php` 里加入：
```php
function hitokoto(){
      $h = htmlspecialchars(file_get_contents('http://api.hitokoto.us/rand'));
      $h = json_decode($h, true);
      return '<a class="hitokoto" target="_blank" title="来源：'.$h[source].'
  投稿：'.$h[author].' @ '.$h[date].'" rel="nofollow">'.$h[hitokoto].'</a>';
  }
```

然后在模板你需要他的一处加入：
```php
  <?php echo hitokoto(); ?>
```

### Javascirpt
Javascript 代码
```javascript
  setTimeout("var elem=document.createElement('script');elem.setAttribute('src','http:/'+'/api.hitokoto.us/rand?enpre=jsc');document.body.appendChild(elem);",50);
  function hitokoto(j){
      if(j.source!=""){
          source = j.source;
      }    else    {
          source='Unknown';
      }
      document.getElementById("hitokoto").innerHTML=j.hitokoto;document.getElementById("hitokoto").title='From: '+source;
  }
```
调用方式：
```html
 <p id="hitokoto">Loading...</p>
```

### JQuery
```javascript
  function hitokoto(d)
  {
    $("#hitokoto").stop().fadeOut(function (){
      $("#hitokoto").html("<a target='_blank' title='分享者：" + d.author + "
  来源：" + d.source + "
  ' href='http://hitokoto.us/view/" + d.id + "'>『" + d.hitokoto + "』</a>");
      $("#hitokoto").stop().fadeIn();
    });
  }
  function genHitokoto(){
    var cats = ['a', 'b', 'c', 'd'], script = document.createElement('script');
    script.type = 'text/javascript';
    script.id = 'hitokotoko_js';
    script.src = "http://api.hitokoto.us/rand?cat=" + cats[Math.floor(Math.random()*cats.length)] + "&amp;encode=jsc&amp;charset=utf-8";
    $("#hitokoto").append(script);
  };
  setInterval(genHitokoto, 10000);
  genHitokoto();
```

调用方式：
```html
  <p id="hitokoto">Loading...</p>
```
